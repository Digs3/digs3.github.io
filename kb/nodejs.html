Node.js (JavaScript runtime)

Official Website

What is NodeJS?

Node.js is a JavaScript runtime environment that runs on your machine instead of in the Web browser.

Development tools such as Gulp, Grunt, and Yeoman are made with Node.js and require it to function.

Use node <.js file> to run javascript locally. You can also just run node to start a REPL shell which will evaluate all javascript you input into it.

What is NPM?

npm is the default package manager for Node.js. It allows users to install Node.js modules that are available on the npm registry/directory.

Modules are installed into the node_modules folder

Modules can be installed globally using the -g modifier.

Some popular modules include:

  • Express (Web development framework)
  • Connect (Extensible HTTP server framework)
  • Socket.io (Server side component for websockets)
  • Pug / Jade (Template engine)
  • Mongo / Mongoose, Coffee-Script, and Redis

package.json

package.json is the packaging format for node.js applications. This file stores metadata relevant to the project such as name, description, version, dependencies, etc. It tells npm how to install your application.

Dependencies in package.json can specified as "dependencies" or "devDependencies"

  • "dependencies" : these packages are required by your application in production
  • "devDependencies" : these packages are only needed for development and testing

Using NPM

  • npm init - creates a basic package.json file
  • npm install - installs all dependencies in package.json to ./node_modules
  • npm install <module> - installs module to the project
    • -save : adds the dependency to package.json as a dependency
    • --save-dev : adds the dependency to package.json as a devDependency
    • -g or --global : global module (available on your PATH, independent of project)
  • npm uninstall --save-dev <module> - uninstalls module and removes it from package.json
  • npm install -g npm - updates npm

    • Noteworthy NPM Packages

    • Babel - Javascript compiler (turns ES6 code into ES5)

    Useful Commands

    List installed project dependencies npm list --depth=0
    List installed global dependencies npm list -g --depth=0
