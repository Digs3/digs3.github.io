Grunt (The JavaScript Task Runner)

Official Website

Grunt is a build system. It is used to preview, test, and build your project. It can concatenate js, prefix css, minify files, and provide a local server with live reload.

Quick Summary

Grunt comes as 2 different tools:

  • grunt-cli : the Grunt Command-line interface
  • grunt module

grunt-cli is installed globally while grunt is installed on a per-project basis.

Grunt is installed and managed via npm, the Node.js package manager.

Install grunt using npm install -g grunt-cli
This installs Grunt and puts the grunt command in your system path

npm install grunt : install the grunt module in your project where you wish to use Grunt

Gruntfile.js - the place where you configure the Grunt tasks for your project

Grunt wrapper function: module.exports = function(grunt) {};
Grunt config object wrapper: grunt.initConfig({});

// Loads the plugin that provides the "uglify" task
grunt.loadNpmTasks('grunt-contrib-uglify');
// Register "uglify" as a default task (runs when you just run grunt)
grunt.registerTask('default', ['uglify']);

Grunt Plugins

  • grunt-contrib-watch run predefined tasks whenever watched file patterns are added, changed or deleted
  • grunt-contrib-connect
