PHP (PHP: Hypertext Processor)

A general-purpose, server-side scripting language suited for web development.

What is PHP

PHP (recursive acronym for PHP: Hypertext Preprocessor) is a widely-used open source scripting language that is especially suited for web development and can be embedded into HTML. With PHP we can generate dynamic page content, access files on the server, collect form data, send and receive cookies, access a database, control user-access, encrypt data, and much more. PHP scripts are executed on the server and PHP files have the extenstion ".php".

Here is an example of some PHP that outputs some text:

        		    <body>
        		      <?php
        		        echo "Hi, I'm a PHP script!";
        		      ?>
        		    </body>

How to use PHP

In order to use PHP we need a web host with PHP and MySQL support or we can install a web server on our own PC, and then install PHP and MySQL.

Setting up a local web server

To quickly set up a PHP development environment on our own PC we can install a web server solution stack which is a combination of software rolled into one package to make it extremely easy to get a local web server running with PHP and MySQL support.

There are many different solution stacks to choose from but here we'll be looking at XAMPP, the most popular PHP development environment.

Using XAMPP

Download XAMPP from the XAMPP website and install it.

Open the XAMPP Control Panel application and click 'Start' for the Apache module. This will start a local web server which can be viewed in the browser by navigating to http://localhost.

PHP Syntax

PHP scripts begin with <?php and end with ?>.

PHP comments begin with //, #, or between /* and */.

Variables

In PHP, variables start with the $ sign. PHP variables are case-sensitive.

PHP is a loosely typed language which means it automatically converts variables to the correct data type.

Scope

PHP has three different variable scopes: local, global, and static.

  • Variables declared outside a function have global scope and can only be accessed outside a function.
  • Variables declared within a function have a local scope and can only be accessed within that function.
  • Variables declared with the static keyword retain their value even after program execution leaves its scope. Static variables are still local to the function they are contained in.

The global keyword is used to access a global variable from within a function.

Global variables are also stored in an array called $GLOBALS[index].

Example code:

        				<?php
        				  $x = 5;  // global variables
        				  $y = 10;
        				  function myTest() {
        				    $z = 20;	// local variable
        				    static $a = 1;	// static variable
        				    $GLOBALS['y'] = $GLOBALS['x'] + $GLOBALS['y'];
        				    echo $a;
        				    $a++;
        				  }
        				?>

Data Types

PHP supports the following data types:

  • String
  • Integer
  • Float
  • Boolean
  • Array
  • Object
  • NULL
  • Resource

echo and print

echo and print are both used to output data. echo has no return value while print has a return value of 1 so it can be used in expressions. echo can take multiple parameters while print can only take one argument.

        				<?php
        				  $txt = "Learn PHP";
        				  echo "<h2>$txt</h2>";
        				  print "<h2>$txt</h2>";
        				  }
        				?>

String Functions

Function Description
strlen(str) Returns the length of a string
str_word_count(str) Returns the number of words in a string
strrev(str) Reverses a string
strpos(str,substr) Returns the position of the first match or FALSE
str_replace(replace,with,str) Replaces some charactders with other characters in a string

Constants

Constants are like variables except that once they are defined they cannot be changed or undefined. Constants are automatically global and can be used across the entire script.

To create a constant, use the define() function.

define(name,value,case-insensitive)
