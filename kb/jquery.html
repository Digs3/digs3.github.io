jQuery (The Write Less, Do More, JavaScript Library)

  • Official Website
  • Quick API Reference

jQuery is a JavaScript library designed to simplify the client-side scripting of HTML. It simplifies document traversal and manipulation, event handling, animation, and Ajax interactions for rapid web development. jQuery is the most popular JavaScript library in use today.

jQuery is represented by the $ symbol in your code.

The $(document).ready() function executes after the DOM is ready

          $(document).ready(function() {
        	// jquery code here
          });
        
          $(function() {
          	// shorthand version
          })

To use jQuery, first use the jQuery function, $, to select an element using CSS selectors.

        $('li');		// select all list items on the page
        $('ul li');		// select list items that are in unordered lists
        $('#header');	// select the element with an ID of 'header'
        $('.person');	// select all elements with a class of 'person'
        $('button[data-panel=panel1]')	// select buttons that have the data-panel attribute set to 'panel1'

Methods

Adding/Removing
        $(li).remove();		// removes the set of matched elements
Showing and Hiding

delay(), hide(), show(), toggle(), slideUp(), slideDown(), slideToggle(), fadeIn(), fadeOut(), fadeToggle()

Animating

animate()

        $('ul .slides').animate({'margin-left': -=720}, 1000);
Getting and Setting Attributes
        $('#header').attr('class');				// get the value of an attribute for the first element 
        $('code').attr('class', 'brush:js');	// set one or more attributes for the set of matched elements
Modify HTML

.html()

        $("#header").html('New Header Title')
Modify Style

.css(), addClass(), removeClass()

        $("#header").css({					// Modify css (does not overwrite previous rules)
        	color: 'red',
        	fontWeight: 'bold',
        	'font-size': '24px'
        })
        
        $("#header").addClass('title');		// Add a class to an element
        $("#header").removeClass('title');	// Remove a class from an element

Events

on(),
click(), focus(), blur(), change()

Use the on() method to bind the same handler to multiple events, otherwise you can use the shorthand versions.

        $("#header").click(function() {
        	console.log("You clicked the header!")l
        });
        
        $("#header").on('mouseenter',function() {
        	console.log("You moused over the header!")l
        });

DOM Traversal

first(), last(), eq(), children(), siblings(), parent(), prev(), next(), closest(), find(), filter(), not()

        $('li')						// returns all list items
        $('li').first()				// returns the first list item
        $('li').last()				// returns the last list item
        $('li').eq(3)				// returns the 4th list item
        $('ul:first').children() 	// returns all children
        $('li:first').siblings() 	// returns all siblings
        $('li:first').parent() 		// returns the parent
        $('li').eq(4).prev() 		// returns the previous sibling
        $('li').eq(4).next() 		// returns the next sibling
        $('li').closest('.list')	// returns the first element with the class '.list' by traversing up
        $('ul').find('li')			// returns all list item descendents
        $('li').filter(':even')		// returns all the even list items (you can also use a filter function)
        $("li").not(":even")		// returns all the odd list items

Testing/Checking

is()

        $("ul").click(function(event) {
          var target = $(event.target);
          if (target.is("li")) {								// checks if the target is a list item
            target.css("background-color", "red");
          }
        });
