Ajax (Asynchronous JavaScript and XML)

  • jQuery.ajax() Documentation

Ajax

Core AJAX Method

$.ajax()

Pass it a configuration object with the data necessary to complete a request.

        // Retrieve the latest version of an HTML page.
        $.ajax({
          url: "test.html",
          cache: false
        })
          .done(function(html) {
            $('#results').append(html);
          });
        
        $.ajax({
            url: 'Demo.json',
            dataType: 'json',
            beforeSend: function() {
                $loader.show();
            }
          }).done(successFunction)
            .fail(errorFunction)
            .always(alwaysFunction);
        
        // Save some data to the server and notify the user once it's complete.
        $.ajax({
          method: "POST",
          url: "some.php",
          data: {name: "John", location: "Boston"}
        })
          .done(function(msg) {
            alert("Data Saved: " + msg);
          });

Convenience Methods

$.get(), $.post(), $.getScript(), $.getJSON

Load Method

$(selector).load(url[, data][, complete])

Load data from the server and place the returned HTML into the matched element.

        
        $bio.load('Bio.html p:first-child', completeFunction);
        
        function completeFunction(responseText, textStatus, request) {
          $bio.css('border', '1px solid #e8e8e8');
          if(textStatus == 'error') {
              $bio.text('An error occurred during your request: ' +  request.status + ' ' + request.statusText);
          } 
        }

jqXHR object

The jQuery XMLHttpRequest (jqXHR) object returned by $.ajax() is a superset of the browser's native XMLHTTPRequest object

Contains responseText and responseXML properties, as well as a getResponseHeader() method.

Promise Methods

.done(), .fail(), .always(), .then()

        var jqxhr = $.ajax( "example.php" )
          .done(function() {
            alert( "success" );
          })
          .fail(function() {
            alert( "error" );
          })
          .always(function() {
            alert( "complete" );
          });
