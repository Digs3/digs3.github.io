Custom Data Attributres

Used to add metadata about any element within an HTML5 page.

They all start with the data- pattern.

They can be read using JavaScript via the getAttribute() method.

Forms

Form Input Types

<input type="type">

Types: search, number, range, color, tel, url, email, date, month, week, time, datetime, datetime-local

Autofocus

Places the focus on a specific form element.

<input type="text" autofocus>

Placeholder

Places placeholder text inside a form field.

<input type="text" placeholder="your name">

Audio

<audio> : a standard way to embed an audio file on a web page

<audio controls>
<source src="song.ogg" type="audio/ogg" />
<source src="song.mp3" type="audio/mpeg" />
Not Supported
</audio>

Multiple sources -> the browser will use the first recognized format.
