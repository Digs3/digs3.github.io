Toolkit

While not exactly necessary, understanding the different tools available for development can greatly improve your workflow.

For starters, get yourself a good text editor for your code, such as the very popular Sublime Text. This is all you need to begin developing for the Web.

Many tools (e.g., Bower, Grunt, Gulp, Yeoman) require Node.js to be installed first. Node.js allows software written in JavaScript to run on your computer.

Many developers use a build system and a package manager in their workflow. Some also use scaffolding tools.

Build systems like Grunt and Gulp can assist you in previewing, testing, and building your project. They can concatenate your js files, prefix your css, minify files, and even provide you with a local server with live reload. Grunt and Gulp are the two most popular build systems. Which, if any, you decide to use is up to you.

Package Managers like Bower will manage all your front-end JavaScript libraries like jQuery, Bootstrap, Angular, etc. They make it easy to install these libraries by taking care of hunting, finding, downloading, and saving the ones you want.

Scaffolding tools like Yeoman can create the entire infrastructure for your site with a single command. This allows you to rapidly get started on new projects. They will scaffold out a new website, write your build configuration (gruntfile or gulpfile), and pull in relevant build task and package manager dependencies. Many different generators are available to fit different needs.

Code

Web sites are built with code. The three core languages of the Web are:

HTML : the markup language used to describe web pages. This is the fundamental language of the Web.

CSS : a style sheet language used to describe the presentation of your web pages.

JavaScript : an object-oriented programming language supported by Web browsers which allows you to add interactivity into your web pages.

Front-End Frameworks

Front-end frameworks like Bootstrap or Foundation provide a platform or foundation on which your site is built.
