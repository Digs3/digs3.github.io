Bower (A Package Manager for the Web)

Official Website

Bower is a package manager. It is to the web browser what NPM is to Node.js. It manages your front-end libraries like jQuery, Foundation, etc.

Installed packages are stored in the bower.json file and placed in the bower_components directory.

Install Bower globally with the command npm install -g bower

Commands

  • bower init: creates a new bower.json config file
  • bower install --save <pkg>: installs a pkg and saves it to bower.json
  • bower install git:/github.com/user/package.git
  • bower uninstall --save jquery: uninstalls pkg and removes it from bower.json
  • bower list --paths: shows all installed packages with paths
  • bower search <pkg>: search for a package in the bower directory
  • bower update <pkg>: update a package to the lastest version
