Handlebars.js (Minimal Templating on Steroids)

Official Website | Tutorial

Handlebars.js is a templating engine for JavaScript. Handlebars is a superset of the Mustache template engine. It can be used to add templates to your HTML page that will be parsed and interpolated with the values from the data you passed to the Handlebars function. This helps you separate the HTML from the dynamic content created by JavaScript.

Handlebars is a compiler that takes any HTML and Handlebars expression and compiles them to a JavaScript function. This function takes an object--your data--and returns a string with the object properties inserted into the HTML.

Installation

Handlebars can be installed with npm or with bower

npm install --save handlebars

bower install --save handlebars

Insert handlebars.js into your site:

        	<script src="handlebars-v4.0.22.js"></script>

Simple Example

        	// HTML + Handlebars expression
        	var myInfo = "<p>My name is {{name}} and I live in {{city}}, {{state}}.</p>";
        
        	// Compile expression into a JavaScript function
        	var template = Handlebars.compile(myInfo);
        
        	// Fill the template with data
        	var data = template({name: "Daniel", city: "Marietta", state: "OH"});
        
        	// Insert the returned html into the page
        	$('derekData').html(data);

Separate Template from JavaScript

Place your HTML + Handlebars expression inside a script tag with the type text/x-handlebars-template and give it an id.

        	<script id="entry-template" type="text/x-handlebars-template">
        	  <p>My name is {{name}} and I live in {{city}}, {{state}}.</p>
        	</script>

HTML Escaping

Use the "triple-stash", {{{, to include any HTML in your expressions. Handlebars will not HTML-escape triple-stashed values.
