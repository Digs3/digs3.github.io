AngularJS

AngularJS is the most popular and powerful web application framework today. Backed by Google and hard to learn, AngularJS tries to make web apps more natural & fluid.
It attempts to solve the problem of complexity behind coding the DOM is JS.

Single Page Application (SPA) - a Web page in which we have a shell page which we can load, on-the-fly, multiple views (mini Web pages)

Building SPA's by scratch includes: DOM Manipulation, Data Binding, Object Modeling, View Loading, Module Loading, History, Caching, Ajax, Routing, etc.
This is built into Angular

Features

  • Data Binding and Dependency Injection
  • Full Testing Environment
  • Deep Linking for Dynamic Pages - encode state application in URL so it can be bookmarked
  • Directives - used to create custom HTML tags that serve as custom elements, also used to decorate
  • MVVM - models talk to viewmodel objecdts through the $scope object - viewmodel can be delivered & rendered by views
  • Routing of Views with built-in routing support
  • jqLite for DOM manipulation (a mini-me of jquery)
  • Templates, History, Factories

Data Binding Expression

Example {{ name }}
This is a Data Binding Expression. It will automatically update to the current value of the given property.
Use the ng-model Directive to add new properties.

Directives - a way to teach HTML new tricks

Allows us to extend HTML using attributes, elements, or comments.

Example: <html ng-app>
ng-app initializes the Angular app.

Example: <input type="text" ng-model="name" />
ng-model adds a property into the scope.

Example: <div class="container" data-ng-init="names=['Dave','Napur','Heedy','Shriva']"> <ul><li data-ng-repeat="name in names">{{ name }}</li> </div>

ng-init allows you to initialize values??
ng-repeat allows you to iterate through an array.
ng-click allows you to specify custom behaviors when an element is clicked.
ng-view complements $route service by including the rendered template of the current route into the main layout file
ng-submit binds an Angular expression to the onsubmit event of the form. (example: <form ng-submit="addTodo()">)

"ng-" Directives are built-in Angular Directive.

Filters

Uses: uppercase, lowercase, limitTo, orderBy, filter, currency, date, number, json

Example:
<input type="text" ng-model="nameText" />
<ul><li ng-repeat="cust in customers | filter:nameText | orderBy:'name'">
{{ cust.name }} - {{ cust.city | lowercase }}

This will allow you to type in a name and instantaneously see the results
