Yeoman (The Web's Scaffolding Tool for Modern Webapps)

Official Website / Tutorial

Yeoman provides the infrastructure for a new Web project through the use of generators. It will scaffold out a new application, writing your build configuration (e.g., Gruntfile, Gulpfile) and pulling in relevant build tasks and package manager dependencies (Bower, npm) that you might need for your build.

Yeoman and its generators are distributed as node modules.

Install Yeoman with npm install -g yo
Install generators with npm install -g generator-angular
List installed generators with yo --generators

Generators
  • generator-webapp - the default Web application generator. It uses Grunt and will scaffold out a project containing HTML5 Boilerplate, jQuery, Modernizr, and Bootstrap
  • generator-angular
  • generator-zf5

subgenerators - used to scaffold further pieces of your application

yo generatorname [appname]: create a new app using the specified generator
yo angular:controller user: generate single components
