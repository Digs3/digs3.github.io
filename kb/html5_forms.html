HTML5 Forms

How to structure HTML forms

The <form> element

The <form> element represents a document section that contains form input fields to submit information to a web server

<form> attributes

  • action
  • method

The <fieldset> and <legend> elements

The <fieldset> element is used to create groups of controls that share the same purpose. The <legend> element is used to label a <fieldset>

Some screen readers will pronounce the legend's content before the label of each control.

Code

        			<form>
        			<fieldset>
        			<legend>Personalia:</legend>
        			Name: <input type="text">
        			Email: <input type="text">
        			</fieldset>
        			</form>

Output

Personalia: Name: Email:

<fieldset> attributes

  • disabled

The <label> element

The <label> element represents a caption for an input field. It can be associated with a control either by placing the control element inside the <label> element, or by using the for attribute.

Code

        			<label>
        			<span>Label</span>
        			<input type="text">
        			</label>

Output

The <input> element

The <input> element creates the data input controls used for web-based forms. How an <input> works depends on its type.

<input type="type">

Types: search, number, range, color, tel, url, email, date, month, week, time, datetime, datetime-local, button, checkbox, file, hidden, image, password, radio, reset, submit, text

Attributes

  • list : the ID of a <datalist> element whose <option> elements will be used as hints and displayed as proposals
  • pattern : a regular expression that the control's value is checked against (used with types: text, tel, search, url, and email.

Other attributes: form, formmethod, x-moz-errormessage

Autofocus

Places the focus on a specific form element.

<input type="text" autofocus>

Placeholder

Places placeholder text inside a form field.

<input type="text" placeholder="your name">

Tabindex

Set the tab order of the element

<input type="text" tabindex="1">

Required

Specifies an input field is required

<input type="text" required>
